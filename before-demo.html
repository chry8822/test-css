📝 ::before와 ::after 차이점

::before - 요소의 앞에 가상 요소 추가

::after - 요소의 뒤에 가상 요소 추가

<div class="example"> ::before <- 여기 실제 내용 ::after <- 여기 </div>

🎯 실제 사용 용도

  • ✅ 장식적 요소: 그라데이션 바, 아이콘, 도형
  • ✅ 호버 효과: 언더라인, 테두리, 오버레이
  • ✅ 상태 표시: 읽음/안읽음, 활성/비활성
  • ✅ 툴팁: 말풍선 꼬리 부분
  • ✅ 배지: 알림 개수, NEW 표시

💡 핵심 포인트

  • content: ''는 필수! (빈 문자열이라도)
  • position: absolute를 쓰려면 부모에 position: relative 필요
  • opacity: 0 + transition으로 부드러운 애니메이션
  • HTML을 수정하지 않고도 시각적 요소 추가 가능
